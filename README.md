# ChatGPT Bulk Delete & Folders

> Bulk delete, organize into folders, search, and export your ChatGPT conversations.
> 100% free. No data collected. Open source.

---

## Why this extension?

ChatGPT gives you no way to bulk delete conversations, no folders, and no instant export.
This extension fixes all of that — completely free.

---

## Features

| Feature | Description |
|---|---|
| ✅ **Bulk Delete** | Select all or filter by date / keyword / model — delete in one click with undo |
| 📁 **Folders** | Create unlimited folders and subfolders. Drag conversations in. |
| 🏷️ **Labels & Tags** | Color labels, custom tags, pin and favorite conversations |
| 🔍 **Search & Filter** | Instant search by title. Filter by date, model, or label. |
| 📤 **Export** | Export to Markdown, TXT, JSON. Bulk export as ZIP. Instant — no 7-day wait. |
| 📊 **Statistics** | Total conversations, most active days, model usage, streaks. |
| ⚙️ **Auto-Cleanup** | Auto-delete or archive conversations older than X days. |
| 💾 **Backup** | Full JSON backup to protect against OpenAI data loss. |
